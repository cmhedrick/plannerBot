Command List
add_schedule - add schedule, ex: "/add_schedule Schedule Title"
schedule - display schedule for chat
add_task - add task, ex: "/add_task Task Title"
update_task_desc - update task description, ex: "/update_task_desc 1 task description"
update_task_datetime - update task date/time, ex: "/update_task_datetime 1 04/26 10pm"
